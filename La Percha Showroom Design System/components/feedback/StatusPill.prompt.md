Canonical status pill — use it everywhere a pedido/pago/publicación shows state so colors stay consistent.

```jsx
<StatusPill status="enviado" />
<StatusPill status="revision" />   // moderación de prendas
<StatusPill status="liberado" />   // pago liberado al vendedor
```

Order/payment: pendiente · pagado · preparando · enviado · entregado · liberado · cancelado. Moderation: revision · aprobada · rechazada · cambios.

Centered placeholder for empty carts, no favorites, no results, or success screens. Pass an icon medallion, title, description and optional action.

```jsx
<EmptyState tone="brand" icon={I.heart(30)}
  title="Todavía no tenés favoritos"
  description="Tocá el corazón en los productos que te gusten para guardarlos acá."
  action={<Button>Explorar</Button>} />
```

Tones tint the icon medallion: neutral, brand, mint, success, warning, error.
